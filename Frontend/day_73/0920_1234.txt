Hyper Text Markup Language (HTML)
HT = 웹에서 자유롭게 오갈 수 있는 링크
M = 텍스트뿐만 아니라 이미지, 영상 등을 표시. 마크업에 사용하는 꼬리표를 태그(tag)라고 함
    ==> HTML = 웹에서 자유롭게 오갈 수 있는 웹 문서를 만드는 언어

[HTML 문서의 기본 구조]
<!DOCTYPE html> : 현재 문서가 html로 작성한 웹 문서라는 뜻
<html>~</html> : 웹 문서의 시작과 끝 <html lang = "ko">
<head>~</head> :  웹브라우저가 웹 문서를 해석하는데 필요한 정보. 웹 브라우저 화면에는 보이지 않음
    <meta charset = "utf-8"> : 문자 세트 등 정보가 들어있는 <meta> 태그, 이 외에도 웹 문서의 키우드, 웹 문서 설명, 웹문서의 소유자나 제작자 등 다양한 문서 정보를 지정
        예, <meta name = "keywords"/"description"/"author" content = "html의 구조"/"html의 구조를 알아봅시다."/"소유자 및 제작자 이름">
    <title>HTML 기본 문서</title> : 문서의 제목을 나타내는 <title>태그. 웹 브라우저의 탭에 나타난다.
<body>~</body> : 실제로 웹 브라우저 화면에 나타나는 내용. 대부분의 html태그가 이 안에서 사용된다. 

[웹 문서 구조를 만드는 시맨틱(semantic) 태그]
왜? : 화면 낭독기 같은 보조 기기에서 사이트의 구조를 이해할 수 있다. 문서 구조가 정확히 나눠지므로 PC나 모바일 등 다양한 화면에서 웹 문서를 표현하기 쉽다. 웹 사이트를 검색할 때 필요한 내용을 정확히 찾을 수 있다.
<header> : 헤더 영역을 나타낸다. 사이트 전체의 헤더 또는 특정 영역의 헤더. 검색 창이나 사이트 메뉴 삽입.
<nav> : 네비게이션 영역. 웹 문서 위치에 영향을 받지 않음. 문서 안에 여러 개 만들 수 있음 (id로 구분)
<main> : 웹 문서의 핵심 내용. 웹 문서마다 다르게 보여주는 내용으로 구성. 웹 문서에 한번만 사용.
<article> : 독립된 웹 콘텐츠 항목. 따로 떼어도 콘텐츠가 되는 내용. <section>태그를 포함 할 수 있다.
<section> : 컨텐츠 영역. 몇 개의 컨텐츠를 묶는 용도로 사용. CSS적용을 위해 묶는 용도(<div> 태그)로 쓰지 말 것.
<aside> : 사이드바 영역. 본문 내용 외에 왼쪽이나 오른쪽 혹은 아래쪽에 사이드바 표시. 필수 요소가 아니므로 필요시에만 사용.
<footer> : 푸터 영역. 사이트 제작 정보나 저작권 정보, 연락처 등. 다른 시멘틱 태그 사용해 다양한 정보 포함.
<div> : 여러 소스를 묶는 용도. 영역을 구별하거나 스타일 적용하기 위해 사용.

[텍스트 태그 알아보기]
h%*6=> 자동으로 h1 ~ h6만든다
<hN>~</hN> : 제목 표시 (N : 1~6, 숫자가 커질 수록 크기가 작아짐)
<p> ~ </p>: 텍스트 단락
<br> : 줄 바꾸기
<blockquote>~</blockquote> : 인용문 넣기. 다른 텍스트보다 안으로 들여 써짐.

<strong></strong> : 중요한 내용이라서 강조할 때
<b></b> : 단순히 굵게 표시

<em></em> : 흐름상 특정 부분을 강조하고 싶을 때
<i></i> : 단순히 이탤릭체로 표시

[목록 만들기]
<ol>, <li></li>, ..., </ol> : 순서 목록. 
    - 각 항목에 숫자가 붙여짐.
    - type 속성 : 순서 목록의 숫자 조정. (1: 숫자, a: 소문자, A: 대문자, i: 로마 소문자, I: 로마 대문자)
    - start 속성 : 목록의 시작 번호 수정

<ul>, <li></li>, ..., </ul> : 순서 없는 목록. 불렛 포인트

<dl>, <dt>이름(제목)</dt>, <dd>값(설명)</dd>, [dt,dd], ..., </dl> : 설명 목록
    - 이름(제목)과 값(설명) 형태로 된 목록
    - 하나의 dt에 여러개의 dd값을 가질 수 있다.

[표 만들기]
표를 만드는 태그 
<caption></caption> : 표 제목
<table></table> : 표 전체
<tr></tr> : 행
<td></td> : 셀
<th></th> : 제목 셀. 열 이름

표의 구조 정의하기
- v표의 구조를 '제목'과 '본문, '요약'부분으로 나눈다.
- <thread>, <tbody>, <tfoot> 태그 사용
- 웹 브라우저 화면에서 보이지 않지만 화면 낭독기나 자바 스크립트 등에서 읽을 수 있다 --> 시각 장애인도 표의 구조를 쉽게 이해할 수 있다
- 표의 본문이 길 경우 자바스크립트를 이용해 제목과 바닥 부분을 고정하고 본문만 스크롤 되도록 할 수 있다.

colspan, rowspan 속성 - 행 또는 열 합치기
<td rowspan = '합칠 셀의 개수'>셀의 내용</td>
<td colspan='합칠 셀의 개수'>셀의 내용</td>

[이미지 삽입하기]
<img src = "이미지파일 경로" alt = "대체용 텍스트"> : 웹문서에 이미지 삽입할 때 사용
    - 웹 페이지에서 사용할 수 있는 이미지 파일은 파일 크기가 크지 않으면서도 화질은 조헤 유지해야 하기 때문에 주로 몇가지 파일 형식만 사용
    - GIF, JPG/JPEG, PNG
width, height 속성 : 이미지 크기 조정

[오디오와 비디오 삽입하기]
<object> 태그 : 음악 파일 뿐만 아니라 동영상이나 자바 애플릿, pdf 파일 등 다양한 개체를 삽입
<object width="너비" height="높이"data="파일"></object>

<embed> 태그 : 대부분 브라우저에서 사용 가능 --> <object>, <audio>, <video> 태그를 지원하는 브라우저에서 멀티 미디어 삽입할 때 사용.
<embed src="파일 경로" width = "너비" height = "높이">
    - 닫는 태그없음

웹 브라우저에서 지원하는 멀티 미디어 파일의 종류
- 비디오 : mp4, webm
- 오디오 : mp3, mp4/m4a

<audio>, <video> 태그
<audio src = "audio file path"></audio>
<video src = 'video file path"></video>

<audio>, <video> 태그 내 속성
<audio src = "medias/spring.mp3" controls></audio>
    - controls : 재생막대 표시
<video src = "medias/salad.mp4" controls width="700"></video>
    - width : 플레이어 너비
그 외
    - autoplay : 자동실행
    - loop : 반복재생
    - muted : 소리제거
    - preload : 페이지를 불러올 때 오디오나 비디오 파일을 어떻게 로딩할 것인가 지정. 사용 가능 값 = auto(기본값), metadata, none
    - width, height : 크기
    - poster = "파일 이름" : <video> 태그에서 사용하는 속성으로 비디오가 재생되기 전까지 화면에 표시될 포스터 이미지
