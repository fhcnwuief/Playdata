function 함수명()|함수명(변수) {
    명령
}

[var를 사용한 변수의 특징]
스코프 : 변수가 적용되는 범위. 지역(로컬) 변수, 전역(글로벌) 변수

지역 변수 : 함수 안에서 선언하고 함수 안에서만 사용. var과 함께 변수 이름 지정

전역 변수 : 스크립스 소스 전체에서 사용. 함수 밖에서 선언하거나 함수 안에서 var없이 선언.

[let과 const]
let : 블록{} 변수. 재할당은 가능하지만 재선언은 할 수 없다. 호이스팅이 없다.

const : 상수. 변하지 않는 값을 선언할 때 사용. 재선언, 재할당 불가.

** 자바스크립트 변수, 이렇게 사용하자 **
- 전역 변수는 최소한
- var 변수는 함수의 시작 부분에서 선언
- for 문에서 카운터 변수는 var보다 let변수로 선언
- ES6를 사용한다면 var보다 let를 사용하는 것이 좋다
